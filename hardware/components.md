## Circuit Notes
- IR sensor sensitivity pot must be calibrated for correct detection
  distance (~2–3cm from eyelid).
- Sensor positioned on spectacle frame facing the eye, angled slightly downward.
- Ambient IR sources (direct sunlight, halogen lamps) can cause false
  triggers — test in varied lighting before use.
